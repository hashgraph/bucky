# Repository for Bucky S3 Client

<img src="bucky.svg" alt="bucky-icon" width="256" height="256">

Bucky is a lightweight Java S3 client with no 3rd party dependencies. It is designed to be minimal and easy to use,
providing a straightforward interface for interacting with Amazon S3 compatible services.

After much searching we could not find a Java S3 client that did not have a large number of dependencies. This client
only uses the Java standard library.
